# ENQI - Product Backlog

---

## Purpose

This document contains all planned features, improvements and product ideas for ENQI.

The backlog is organized by business value rather than implementation order.

Features remain in this document until they are officially planned inside a project sprint.

---

# Priority Levels

🔴 Critical

Core features required for the first production release.

🟠 High

Features that significantly increase customer value.

🟡 Medium

Important improvements that can be implemented after the MVP.

🟢 Low

Future improvements and optional features.

💡 Research

Ideas under evaluation that require further technical or business analysis.

---

# 🔴 Critical

## Workspace Management

Status

Planned

Description

Organizations organize their knowledge through configurable Workspaces instead of fixed modules.

Business Value

Provides flexibility for organizations from different industries.

---

## Role Based Access Control (RBAC)

Status

Planned

Description

Permission management based on roles, departments and organizational hierarchy.

Business Value

Guarantees security and LGPD compliance.

---

## Intelligent Document Processing

Status

Planned

Description

Automatic classification, OCR, metadata extraction and indexing.

Business Value

Transforms documents into searchable organizational knowledge.

---

# 🟠 High

## Smart Organization Bootstrap

Status

Planned

Description

Automatically prepares an optimized workspace structure based on the organization's profile.

Business Value

Reduces onboarding friction and delivers value from day one.

---

## AI Knowledge Search

Status

Planned

Description

Natural language search across organizational knowledge respecting user permissions.

Business Value

Transforms documents into intelligent answers.

---

# 💡 Research

## Organization Identity Service

Status

Under Evaluation

Description

Identify organizations using official registration numbers (CNPJ, EIN and equivalent identifiers).

Possible Benefits

- Automatic organization profile
- Reduced onboarding time
- Industry detection
- Smart Organization Bootstrap
- Better user experience

Open Questions

- Multi-country support
- Public APIs
- Data providers
- Caching strategy
- Offline behavior
